◆ MAX_PAYLOAD_SIZE

#define MAX_PAYLOAD_SIZE   144

System defines Maximum size of fragmented network frames and fragmentation cache.

Note
: This buffer can now be any size > 24. Previously need to be a multiple of 24.
: If used with RF24Ethernet, this value is used to set the buffer sizes.
Examples
helloworld_rx_advanced.ino, and helloworld_tx_advanced.ino.

◆ MAIN_BUFFER_SIZE

#define MAIN_BUFFER_SIZE   (MAX_PAYLOAD_SIZE + FRAME_HEADER_SIZE)

The size of the main buffer. This is the user-cache, where incoming data is stored. Data is stored using Frames: Header (8-bytes) + Frame_Size (2-bytes) + Data (?-bytes)
